## Información

Aplicación que se encargar de obtener las categorías y aplicaciones desde el rss de Appel.
Una vez obtenida la información, se almacena en la base de datos local para poder hacer consultar cuando
no se hay acceso a información

## Funcionalidades

* Splash screen
* Vista de categorías
* Vista de aplicaciones por categorías
* Vista de información de categoría

## Caracteristicas

* Implementación de la biblioteca Volley de Google para consultas de servicios web
* Implementación de cache de imagenes con la biblioteca Volley de Google
* Implementación de animaciones
* Implementación de diseño para tablets y telefonos
